# Global Data Jobs Market Analysis using Python

## Project Objective

This project analyses global data-related job postings to identify patterns in salary trends, remote work opportunities, hiring demand, company-based salary differences, and country-level job market behaviour using Python.

## Tools & Libraries Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Skills Demonstrated

* Data Cleaning
* Data Filtering
* Data Grouping
* Exploratory Data Analysis (EDA)
* Salary Analysis
* Remote Work Analysis
* Data Visualization

## Key Insights

* Senior Data Scientist roles had the highest salaries
* Data Engineering roles dominated remote opportunities
* The United States had the highest number of remote jobs
* Entry-level Data Science roles showed strong salary growth
